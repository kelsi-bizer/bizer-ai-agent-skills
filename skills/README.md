# Skills

One folder per skill, each self-contained: the installable `SKILL.md`, a
README, and (where applicable) an `evaluation/` fixture with expected
findings so every revision can be judged by transcript.

| Skill | What it does | Version |
| --- | --- | --- |
| [fraud-watch](fraud-watch/) | Review bank or card transactions from any source and surface possible fraud, the way an analyst would. Calibrated on 2.65M labeled transactions. | 1.0.0 |

## Install

See the [repo README](../README.md) for the one-command Claude Code install,
the copy-a-folder path for any Skills-capable agent, and the Bizer app import.
